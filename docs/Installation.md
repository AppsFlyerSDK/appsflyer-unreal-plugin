# Installation

1. In your game’s root directory, create the **Plugins** directory
2. Copy the **AppsFlyer** folder into the **Plugins** dirrectory to create the following file hierarchy:

```
  MyProject
  ├── Plugins
          └── AppsFlyerSDK
              └── AppsFlyerSDK.uplugin
```
